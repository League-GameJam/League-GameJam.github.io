The Contest:

When the countdown reaches 0, a link will become available to a GitHub Classroom assignment. The assignment is an empty repository that registered contestants will use to submit their code. Contestants will then have two weeks to build their game using either Java or Processing.

In addition to the GitHub link becoming available, a theme will also be announced. All submitted games must fit within this theme. Once the deadline is reached, the GitHub Classroom assignment will become locked and no further changes to the game will be permitted. The games will then be judged and winners will be chosen based on the following catagories:

  1. Best Overall Experience
  2. Best Interpretation of Theme
  3. Best Graphics
  4. Best Use of Input Devices (mouse, keyboard, gamepad, etc...)
  5. Best Game from a Beginner Coder

Schedule of Events:

  • Aug 1st, 6:30pm: Registration Ends and Theme is Announced
  • Aug 1st, 6:30pm - 8:00pm: Optional Kick-off Event

Official Rules:

  • Contestants must be a currently enrolled student of The League of Amazing Programmers
  • Games must fit within the theme
  • Games must be completely written in Java or using the Processing IDE
  • Games must be appropriate and family friendly
  • All code must be writtin after the start time
  • Contestants can participate individually or as a team
  • Code must be 100% written by the contestants
  • All game assets must be original or used with permission. Plagarism will not be tolerated
  • All game changes must be submitted before the ending deadline
